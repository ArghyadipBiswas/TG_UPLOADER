# TG_UPLOADER
This tool can upload files to telegram from links.

Setup tutorial 
```
git clone https://github.com/ArghyadipBiswas/TG_UPLOADER && cd TG_UPLOADER && chmod 777 * && ./install.sh
./main.sh
```
